# Oracle SQL HR Analytics Project 

## Description 
This project analyzes Oracle HR sample schema using SQL. 

## Tools 
- Oracle SQL
- HR sample schema
  
## Key Tasks 
- Employee and department analysis 
- Salary aggregation and filtering
- LEFT JOIN and NULL handling
- Subqueries and aggregate functions
  
## Outcome 
Identified departments with high average salaries and employees earning above company average.
